처음에는 제목을 "C++ 개발자가 Java에 적응하기까지"라고 지었다가 지금의 제목으로 고쳤다. 아직 Java에 적응을 마치지 못했기 때문에 "적응하기까지"라는 표현은 틀렸다.
팀과 업무와 사용 언어가 바뀐지 1년이 넘었건만 아직 적응 중이다. 어쩌면 영영 적응을 마칠 일이 없을지도 모른다.

내 지난 경험을 조금 풀어보겠다. Java를 처음 접했을 때가 대학교 2학년이었다. Java 2가 나온지 오래되지 않은 시점이었다.
이후 한두 과목에서 Java를 사용했고, 졸업 후 한동안 개발과 동떨어진 일을 하다가 거의 5년 만에 짧은 인턴 기간 중 다시 접할 기회가 생겼다. 이때는 Java 5였다고 기억한다.

C++에 관한 기억은 조금 더 단순한데, 자료 구조 수업에서 사용한 이후로 죽 손댈 일이 없다가, 역시 인턴 기간에 잠시 기억을 되살릴 시간을 가졌다.
얼마 후 지금의 회사에서 검색 엔진 개발팀에 들어가면서 10년 넘게 C++을 곁에 두고 살았다.
C++ 개발자라면 잘 알다시피, 그사이 이 역시 C++98부터 곧 발표될 C++23까지 새 표준을 통해 꾸준히 변해왔다.
